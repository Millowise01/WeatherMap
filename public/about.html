About Weather Dashboard

The Weather Dashboard is a user-friendly application designed to provide real-time weather information and 5-day forecasts for any city worldwide. Powered by the OpenWeatherMap API, it offers features like:

  • Search for current weather and forecasts by city name.
  • Sort forecast data by temperature (ascending or descending).
  • Filter forecasts by weather conditions (e.g., Clear, Rain, Clouds).
  • View recent searches in the Search History page.

This application was built using HTML, CSS (Tailwind CSS), JavaScript, and Node.js with Express for the backend. It is containerized with Docker and deployed with a load balancer for scalability.
